# Availability_of_Data_and_Materials

This repository aims to provide the data supporting the article entitled **"Open-source_slicing_program_based_on_filament_extrusion_model_localized_controlled_flexibility_and_evaluation_of_cyclic_loading"**

The repository includes the raw data obtained from the cyclic tests, the Python script used for data processing, and the resulting processed datasets employed in the analyses presented in the article.

## Step-by-step guide

**1-** Access: IJAMT_Availability_of_Data_and_Materials -> Cycling_tests;

**2-** Inside /Cycling_tests/, the Python script (named data_filter.py) and the folders organized according to the gap percentage levels are available;

**3-** The data_filter.py script is fully commented, describing each logical stage of its operation;

**4-** Within the folders corresponding to the gap percentage levels (20percent to 60percent), subfolders associated with the respective number of gaps levels (from 1 to 4 gaps) are available;

**5-** Inside each folder corresponding to a specific number of gaps, two subfolders named input and output are provided;

**6-** The input folder contains three .txt files, each corresponding to the raw data obtained from the cyclic tests for one replicate of that specific configuration, which serve as input files for the Python script. The output folder contains the corresponding processed output data generated by the Python script.
